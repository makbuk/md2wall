Скрипт сам установит Pillow, сгенерирует PNG картинку 1920×1080 и поставит её как обои через KDE API. 
Редактирование: меняешь content.md → запускаешь bash update-wallpaper.sh → обои обновились.

Автообновление каждые 10 минут через cron:
crontab -e
# Добавить строку:
*/10 * * * * bash /путь/до/update-wallpaper.sh
